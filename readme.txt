---Добро пожаловать в ОлимпКалендарь!---
Приложение предназначено для удобной сортировки и слежке за олимпиадами.
Благодаря кнопкам вы можете управлять таблицей - добавлять, удалять и изменять сохраненные олимпиады.
Для создания новой олимпиады вам надо выбрать соответсвующие статусы в каждой из четырех категорий, все они изменяемы:
текстовые путем нажатия на галочку, ввода собственного варианта и повторного нажатия галочки, дата, путем нажатия на
галочку слева сверху и последующего нажатия на табло с датой и выборе подходящей даты во всплывающем календаре.
Помимо этого пользователь может удалить любую из олимпиад путем нажатия на один из его элементов или вводом id олимпиады.
В приложении есть календарь, нажав на любую дату можно получить сведения о олимпиаде в эту дату проводиммой, если таковая
есть. Пользователь может получить полную информацию путем выбора любой из колонок олимпиады и нажатия на кнопку посмотреть.
Наконец - пользователь может очистить все, или раскрыть все олимпиады доступные ему.
Все данные таблицы хранятся в файле results.csv
